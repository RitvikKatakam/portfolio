3d Solar System

This project is an interactive 3D model of the solar system built using Three.js, HTML, and CSS. It visually represents all 8 planets revolving around the Sun, each following its own orbit in real-time. The user interface is designed with simple HTML and styled using CSS to enhance the experience, while Three.js powers the 3D rendering and animation. The simulation provides a basic yet engaging visualization of planetary motion and solar system structure.

AI Based Smart PDF Reader

This project is an intelligent PDF reader built with PyPDF for reading document content, Streamlit for creating an interactive web interface, and RAG (Retrieval-Augmented Generation) to enable smart question-answering. Users can upload a PDF, ask questions about its content, and receive accurate answers powered by retrieval and language models. It enhances document understanding and makes reading large files easier and more interactive.

CreditCardFraudDetection

This project aims to detect fraudulent credit card transactions using machine learning. It analyzes patterns in past transaction data to identify suspicious or unusual activity. By training a model on labeled data (fraud or not fraud), the system can predict and alert when a new transaction seems risky. This helps in preventing financial loss and increasing the security of online payments.
